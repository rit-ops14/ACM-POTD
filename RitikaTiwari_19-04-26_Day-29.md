#Day 29- ( fibonacci number)

##Date - 19/04/26

##Problem -  fibonacci number

##code(C++)- '''class Solution {
public:
    int fib(int n) {
        if(n<=1) return n;
        return fib(n-1)+fib(n-2);
    }
};'''
